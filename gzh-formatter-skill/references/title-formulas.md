# 标题公式参考

## 核心格式

```
AI 早报｜{今日头条（中文）}
```

### 规则

| 元素 | 规则 |
|------|------|
| **前缀** | 固定为 `AI 早报｜` |
| **头条** | 取所有新闻中最重要的第 1 条，标题翻译为中文（如原为英文） |

### 头条选择策略

1. 如果数据有 `score` 字段 → 取 score 最高者
2. 如果无 score → 取出现顺序的第 1 条
3. 如果头条标题是英文 → 翻译为中文嵌入标题

### 示例

```
AI 早报｜Midjourney 进军医疗影像
AI 早报｜DeepSeek 发布百万上下文模型
AI 早报｜AI 教父炮轰 xAI 为"失败品"
AI 早报｜OpenAI 发布 GPT-5 Lite
```

## 备选／兜底

如果当天采集数据质量不佳（标题不完整等），使用通用格式：
```
AI 早报｜今日 AI 动态速览
```

## 注意事项

- 头条一定要翻译成中文，不要中英混杂
- 如果头条超长（>30 字），做截断加 `...`
