# 治愈系 OPC 落地手册 — 从 0 到 1

> 路径：A（数字产品驱动）+ C（IP 内容驱动）
> 模式：副业 2-3h/天
> 平台：小红书（种草引流）+ 闲鱼（成交交付）

---

## 一、IP 定位与风格锚定

### 1.1 IP 角色设定

在出图之前，先确定一个可识别的"主角"，所有内容围绕它展开。

**推荐方案 — 选一个你喜欢的方向：**

| 方向 | 示例 | 适合你吗 |
|------|------|----------|
| 慵懒小动物 | 一只永远在打盹的橘猫、一只捧着热可可的兔子 | AI 出图友好，风格统一容易 |
| 微缩世界 | 迷你小屋、指尖上的咖啡馆、瓶中四季 | 视觉冲击力强，小红书爆款率高 |
| 自然治愈 | 晨雾森林、窗台花园、雨后小径 | 情绪共鸣强，适合实拍+AI 混合 |
| 日常温暖 | 一杯热茶的书桌、黄昏的窗边、柔软的被子 | 最贴近生活，容易引发"想要"的冲动 |

> **执行动作**：本周内选定1个方向，不要贪多。1个方向做深 > 3个方向做浅。

### 1.2 风格关键词（AI 出图 Prompt 基础）

选定方向后，提炼 5-8 个风格关键词，作为所有 Prompt 的基调：

```
# 示例（微缩世界方向）
cozy, miniature, warm lighting, pastel tones, soft focus,
kawaii, detailed, healing atmosphere, dreamy, gentle
```

**色彩规范**：限定 3-5 种主色调，确保系列感。

| 情绪 | 推荐色板 |
|------|----------|
| 温暖治愈 | #F5E6D3 / #E8C4A0 / #D4A574 / #F0D9C6 |
| 清新自然 | #D5E8D4 / #A8D8B9 / #F0F7E6 / #C8E6C9 |
| 柔和梦幻 | #E8D5E0 / #C9B1D0 / #F0E0F0 / #D4B8E0 |
| 夜晚安宁 | #2C3E6B / #3D5A99 / #5B7EC2 / #8FAADC |

## 二、出图规范

### 2.1 图片尺寸大全

> 这是最容易忽略但最关键的细节，尺寸不对直接影响展示效果。

#### 小红书内容图

| 类型 | 尺寸（px） | 比例 | 用途 |
|------|-----------|------|------|
| 竖图（推荐） | 1080 × 1440 | 3:4 | 最通用，信息流展示面积最大 |
| 方图 | 1080 × 1080 | 1:1 | 适合对比图、拼图 |
| 横图 | 1440 × 1080 | 4:3 | 不推荐，信息流占比小 |

#### 小红书封面图

| 类型 | 尺寸（px） | 说明 |
|------|-----------|------|
| 标准封面 | 1242 × 1660 | 3:4，最推荐 |
| 方封面 | 1080 × 1080 | 1:1 |

#### 数字产品 — 手机壁纸

### 2.2 AI 出图工作流

#### 工具选择

| 工具 | 适合场景 | 月费 | 学习曲线 |
|------|----------|------|----------|
| Midjourney | 风格化插画、梦幻场景 | $10/$30 | 低 |
| Stable Diffusion (WebUI) | 可控性高、本地运行、免费 | 免费（需显卡） | 中高 |
| DALL·E 3 | 对话式生成、理解中文 | ChatGPT Plus $20 | 最低 |
| 通义万相 / 文心一格 | 中文理解好、合规 | 部分免费 | 低 |

> **起步推荐**：Midjourney（出图质量最稳定）或 DALL·E 3（最容易上手），等跑通后再考虑 SD 做精细化控制。

#### Midjourney 出图参数速查

```
# 壁纸（竖图 9:16）
/imagine prompt: [你的描述] --ar 9:16 --v 6.1 --style raw

# 头像（方图 1:1）
/imagine prompt: [你的描述] --ar 1:1 --v 6.1 --style raw

# 小红书内容图（3:4）
/imagine prompt: [你的描述] --ar 3:4 --v 6.1 --style raw

# 保持角色一致性（重要！）
# 方法1: 使用 --sref 风格参考
/imagine prompt: [描述] --sref [图片URL] --sw 500

# 方法2: 使用 --cref 角色参考
/imagine prompt: [描述] --cref [角色图片URL] --cw 100
```

#### 出图后处理清单

1. **分辨率检查**：AI 出图通常 1024×1024，壁纸需要放大 → 用 Topaz Gigapixel AI 或 waifu2x 放大到目标尺寸
2. **色彩微调**：统一色调，确保系列感 → 用手机 App（Snapseed/VSCO）或 PS 批量调色
3. **水印添加**：小红书发布前加轻度水印（防搬运，但不要太影响观感）
4. **裁切适配**：同一张图可能需要裁出壁纸版、头像版、封面版

### 2.3 实拍照片工作流

| 步骤 | 操作 | 工具 |
|------|------|------|
| 拍摄 | 日系/胶片滤镜，光线偏暖偏柔 | 手机相机 + 自然光 |
| 筛选 | 每次拍 20-30 张，选 3-5 张 | 相册 |
| 调色 | 统一色调，降对比、提暗部、偏暖 | VSCO / Lightroom Mobile |
| 裁切 | 按用途裁到对应尺寸 | Snapseed |
| 叠加（可选） | AI 出图元素与实拍合成 | PS Express / Canva |

## 三、内容生产 SOP

### 3.1 每日内容节奏（2-3h/天）

| 时间 | 任务 | 时长 | 产出 |
|------|------|------|------|
| 30min | AI 出图 / 实拍 | 0.5h | 5-10 张原始图 |
| 30min | 筛选 + 后期修图 | 0.5h | 3-5 张成品图 |
| 30min | 撰写文案 + 排版 | 0.5h | 1 条小红书笔记 |
| 30min | 互动运营 + 数据记录 | 0.5h | 评论区互动、数据表更新 |
| 30min | 数字产品打包（非每日） | 0.5h | 每周 1-2 个产品包 |

> **核心原则**：每天至少发 1 条小红书，连续 30 天不断更。前 30 天是"养号期"，算法需要学习你的内容标签。

### 3.2 小红书发布规范

#### 笔记类型与模板

**类型 1 — 壁纸分享（引流主力）**

```
标题格式：
- "治愈系壁纸 | [情绪词] [场景描述]"
- 示例："治愈系壁纸 | 下雨天窝在沙发里的温柔"

图片：3-6 张壁纸图，第一张做封面，加文字标题
文案模板：
  ✨ 今天的心情是 [情绪]
  🎨 一组 [风格] 壁纸送给你
  💫 保存方式：长按图片→保存到相册
  💌 想要更多治愈壁纸？主页自取～

  #治愈系壁纸 #手机壁纸 #治愈 #晚安壁纸 #小众壁纸
```

**类型 2 — 头像分享**

```
标题格式：
- "治愈头像 | [风格词] [数量]张"
- 示例："治愈头像 | 柔软奶油风 8 张"

图片：4-8 张头像，2×4 拼图做封面
文案模板：
  🌿 换个头像换种心情
  🍃 [风格] 系列头像，温柔又特别
  💾 长按保存即可使用

  #治愈头像 #小众头像 #女生头像 #温柔头像 #治愈系
```

**类型 3 — 过程分享 / 幕后（涨粉利器）**

```
标题格式：
- "从 0 到 1 | 一张治愈壁纸的诞生"
- 示例："从 0 到 1 | 用 AI 画一个温暖的小角落"

图片：Prompt 截图 + 过程图 + 成品对比
文案模板：
  🎬 很多宝子问我图怎么做的
  📝 今天分享一下我的创作过程
  1️⃣ 先想一个 [情绪/场景]
  2️⃣ 用 [工具] 生成初版
  3️⃣ 调色 + 裁切 → 成品！
  💡 有问题评论区见～

  #AI绘画 #治愈系 #创作过程 #Midjourney
```

#### 发布时间

| 时段 | 时间 | 适合发布 |
|------|------|----------|
| 早高峰 | 7:30 - 8:30 | 壁纸类（换新一天的心情） |
| 午休 | 12:00 - 13:00 | 头像类、轻松内容 |
| 晚高峰 | 20:00 - 22:00 | 所有类型（流量最大） |
| 深夜 | 22:00 - 23:00 | 治愈/晚安类壁纸（情绪需求最强） |

> **建议**：固定在 21:00 发布，前期每天 1 条，稳定后可增加到 1-2 条。

### 3.3 文案关键词库

标题和文案中反复使用以下关键词，强化账号标签：

```
高频词：治愈、温柔、柔软、安静、温暖、舒服、惬意
```

### 5.2 闲鱼店铺设置

| 项目 | 建议 |
|------|------|
| 店铺名 | 与小红书同名，方便搜索 |
| 签名 | "原创治愈系壁纸/头像 | 高清无水印 | 即拍即发" |
| 商品分类 | 壁纸包 / 头像包 / 套装 / 限定 |
| 发货方式 | 自动发货（百度网盘链接），设置自动回复 |

### 5.3 闲鱼自动发货设置

```
1. 图片包上传到百度网盘，生成分享链接
2. 闲鱼商品设置 → 自动发货 → 粘贴网盘链接和提取码
3. 购买后系统自动发送，无需人工干预
4. 每次更新产品包后记得更新链接
```

### 5.4 引流路径

```
小红书笔记 → 评论区引导"想要无水印高清版？主页简介有入口"
             → 简介放闲鱼搜索关键词（如"春日花园壁纸包"）
             → 闲鱼成交
             → （可选）引导加微信，沉淀私域，后续推新品和定制服务
```

> **注意**：小红书不允许直接放外部链接，用"主页简介引导 + 闲鱼关键词搜索"是最安全的方式。

| AI 图同质化严重 | 靠 IP 角色和统一色调做差异化，不是卖图而是卖"系列感" |
| 小红书限流/封号 | 不硬广、不违规引流，前期纯做内容价值输出 |
| 闲鱼盗图/搬运 | 加水印、低分辨率预览、付费版高清无水印 |
| 副业时间不够 | 优先保证每天 1 条笔记，产品包可以每周做 1 次 |
| AI 工具政策变化 | 不要只依赖一个工具，掌握 2-3 个备选 |
| 变现慢，动力不足 | 关注收藏量而不只是销量，收藏=潜在需求，变现只是时间问题 |

## 十、第一步行动

> 读完这份文档后，你现在就可以做的第一件事：
