## 二、组件规范

### 2.1 外标题区（公众号平台自动显示）

```
位置：页面最顶部，白色背景
结构：
  - 外标题（22px/900/黑色）：品牌名｜要点新闻
  - 作者信息（12px/灰色）：公众号名 · 日期 时间
约束：由公众号后台设置，不在 HTML 内生成
```

### 2.2 深色容器（报头）

```
背景：#0B1220
内边距：28px 24px 24px
上边距：16px（与外标题区隔开）

内容层次（从上到下）：
1. 副标题（14px/500/#64748B）：品牌 slogan
2. 日期行（flex 布局）：
   - 日期（13px/700/等宽/青色）
   - 星期（12px/#64748B）
   - 渐变分割线（flex:1/青色渐变透明）
   - 数据徽章（flex 布局）：
     · 今日 N 条（数字青色加粗）
     · 国内 N（数字青色加粗）
     · 国际 N（数字蓝色加粗）
3. 内容速览区块：
   - 背景：rgba(255,255,255,0.03)
   - 圆角：12px
   - 内边距：16px 18px
   - 标题行：📋 + "内容速览"（13px/800/青色）
   - 条目列表：青色圆点 + 标题（13px/#CBD5E1）
```

### 2.3 分类栏目条

```
布局：flex + align-items:center
结构：
  - 色块竖条（4px×16px/圆角 2px/青色或蓝色）
  - 栏目标题（15px/900/黑色）：国内 · DOMESTIC / 国际 · GLOBAL
  - 分割线（flex:1/1px/#E2E8F0）
  - 条数标签（11px/#94A3B8）：N 条
```

### 2.4 资讯条目

```
布局：flex + align-items:flex-start
内边距：14px 0
分割线：border-bottom:1px solid #F1F5F9（最后一项无分割线）

结构：
  - 编号（13px/800/青色或蓝色/宽度 24px）：01/02/03/04
  - 内容区（flex:1）：
    · 标题（15px/700/黑色/行高 1.5）
    · 摘要（13px/#475569/行高 1.7）
    · 来源（11px/600/#64748B）：媒体名
```

### 2.5 CTA 容器

```
背景：linear-gradient(135deg,#0B1220,#1E293B)
圆角：14px
内边距：22px 20px
对齐：text-align:center

内容：
  - 互动引导（14px/#CBD5E1）：觉得有用，欢迎点赞、评论、转发
  - 品牌落款（10px/#475569/字间距 1.5px）：AI DAILY · AI 早报
```

## 三、公众号平台约束

### 3.1 必须遵守

| 规则 | 说明 |
|------|------|
| 样式全内联 | 所有 CSS 写在 `style` 属性内，不用 `<style>` 标签 |
| 文字节点包裹 | 每个文字节点用 `<span leaf="">文字</span>` 包裹 |
| 装饰元素占位 | 空元素（色块、分割线）内放 `<span leaf=""><br></span>` |
| 禁用标签 | 不用 `<style>/<script>/<div>/class/id` |
| 禁用样式 | 不用 `position:fixed/absolute/sticky`、`float`、`@media/@keyframes`、`display:grid`、CSS 变量 |
| 可用布局 | `display:flex`（有限支持）、`linear-gradient`、`border-radius`、`box-shadow` |

### 3.2 校验脚本

```bash
# 生成后必跑校验
python3 <SKILL_ROOT>/scripts/validate_gzh_html.py <产物.html>

# 检查项：
# - 禁用标签/属性/样式 → 0 ERROR
# - <span leaf> 包裹 → 0 WARNING
# - 半角标点 → 0 WARNING
```

## 四、内容结构

### 4.1 标准骨架

```
1. 外标题（公众号后台设置）
2. 深色容器（报头）
   - 副标题
   - 日期行 + 数据徽章
   - 内容速览
3. 国内板块
   - 栏目条
   - 条目 × N
4. 国际板块
   - 栏目条
   - 条目 × M
5. CTA 容器
```

### 4.2 数据规范

| 字段 | 规范 |
|------|------|
| 总条数 | ≤10 条（国内 + 国际） |
| 速览条数 | 3-5 条（从当日条目中精选） |
| 摘要长度 | ≤30 字一句话 |
| 编号格式 | 两位数字 `01/02/03/04` |
| 栏目命名 | `国内 · DOMESTIC` / `国际 · GLOBAL` |
