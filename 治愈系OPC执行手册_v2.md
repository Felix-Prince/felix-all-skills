## 二、出图 SOP — 每周产出节奏

### 2.1 周产出目标（试水期）

| 产出 | 数量 | 说明 |
|------|------|------|
| AI 原始候选图 | 40-60 张 | 3-5 个主题，每主题跑 4-8 次 |
| 精选成品图 | 8-12 张 | 筛选率 10-20%，只留最好的 |
| 小红书笔记 | 5-7 篇 | 每天 1 篇，覆盖不同类型 |
| 产品包 | 1-2 个 | 壁纸包 / 手帐模板 / 数字艺术 |
| 闲鱼商品 | 1-2 个 | 新上架或更新 |

**风格变体关键词（叠加到基础 Prompt 上）**

```
水彩治愈感：  watercolor wash, soft blending, gentle brushstrokes
胶片质感感：  film grain, kodak portra tones, vintage warmth
极简留白感：  minimalist composition, negative space, clean
日系动漫感：  anime style, Studio Ghibli, soft shading
微缩世界感：  miniature, tiny world, detailed small scene
柔粉梦幻感：  pastel dream, soft pink haze, ethereal glow
```

**生图操作流程**

```
Step 1: 复制 Prompt 模板，填入本周主题的场景描述和情绪关键词
Step 2: 在 Midjourney Discord 频道执行 /imagine
Step 3: 等待 4 张候选图生成（约 60 秒）
Step 4: 选择最好的 1-2 张 → U（Upscale 放大）
Step 5: 如果不满意 → V（Variation 变体）或重新调整 Prompt
Step 6: 每个主题重复 2-3 次，总共得到 4-6 张候选
Step 7: 主题间切换时，更换场景描述但保持风格关键词一致
```

### 2.4 人工增强 SOP（每张 10-15 分钟）

```
===== Step 1: 调色（Snapseed） =====

操作清单（按顺序）：
  1. 亮度 +5~10        → 提亮，增加通透感
  2. 氛围 +10~20       → 增加暖色晕染
  3. 暖色调 +5~10      → 统一暖色调
  4. 颗粒 +5~8         → 胶片质感，降低 AI 味
  5. 暗角 +10          → 轻微暗角，增加聚焦感
  6. 对比度 -5         → 降低对比，更柔和

治愈系调色核心原则：
  "暖、柔、透" — 比原图稍亮、稍暖、稍柔，不要太重

===== Step 2: 文案叠加（Canva） =====

字体选择：
  - 中文：思源宋体细 / 汉仪晓波折纸体 / 阿里普惠体细
  - 日文：Noto Sans JP Light
  - 英文：Playfair Display / Cormorant Garamond

文字位置原则：
  - 找画面留白区域（天空/墙面/桌面空白处）
  - 不遮挡画面主体（人物/猫咪/花朵）
  - 字号小而精致，不喧宾夺主
  - 最多 1-2 行短句

文案叠加排版参考：
  ┌──────────────────────────────┐
  │                              │
  │          [画面主体]           │
  │                              │
  │                              │
  │    「慢慢来，比较快」         │ ← 留白区叠加文案
  │                              │
  └──────────────────────────────┘

===== Step 3: 安全区检查 =====

手机壁纸：
  - 顶部 80px 区域 → 无关键内容（避开状态栏/刘海）
  - 底部 120px 区域 → 无关键内容（避开dock/手势条）

头像（1:1）：
  - 中心 600×600px → 包含完整主体
  - 四角会被裁为圆形 → 不要放重要内容在角落

===== Step 4: 导出与命名 =====

导出参数：
  - 格式：JPG 质量 90-95%
  - 壁纸类：PNG（无损，画质优先）
  - 贴纸类：PNG 透明底

命名规则：
  [主题]_[序号]_[尺寸]_[版本].[格式]

  示例：
  spring_rain_01_1080x1920_text.jpg     （带文案版）
  spring_rain_01_1080x1920_clean.jpg    （纯净版）
  spring_rain_01_2560x1440_clean.jpg    （电脑版）
  spring_rain_01_2480x3508_print.jpg    （可打印版）
```

## 七、图片防白嫖策略（核心）

> 数字产品变现的核心矛盾：展示效果要好，但不能让用户直接保存就够用。
> 核心原则：不是让用户"完全不能用"，而是让免费版"能用但不舒服"——模糊、有水印、缺纯净版、缺全套。
> 付费买的不是图片本身，而是 **高清+无水印+纯净版+完整套装** 的体验升级。

### 7.1 分层防护：免费版"能用但不好用"

```
小红书发布的图（免费版）：       付费版的图：
  - 分辨率 72 DPI                - 分辨率 300 DPI
  - 尺寸 540×960（缩小50%）      - 尺寸 1080×1920（原始尺寸）
  - 有水印                        - 无水印
  - 有文案叠加                    - 纯净版 + 带文案版 双版本
  - 色彩略降饱和                  - 原始色彩

效果：用户长按保存到的只是低清版，设为壁纸会模糊/像素化，想要高清无水印就得付费。
```

### 7.2 水印策略：美观与防护的平衡

```
原则：水印不影响观感，但截不掉

位置选择（按推荐度排序）：
  ✅ 画面留白处（天空/墙面/桌面空白），半透明 15-20%   ← 最推荐
  ✅ 画面底部 1/5 处，与文案叠加融合                    ← 次推荐
  ❌ 正中央大水印（太丑，影响内容质量，反而降流量）      ← 不要用

水印内容：
  - 账号名或小红书号（如 @治愈系小角落）
  - 不要只放 logo，放账号名才有引流效果

水印样式：
  - 字体细小、颜色取画面中的浅色、透明度 15-20%
  - 手写字体比黑体更融入画面

Canva 操作步骤：
  文字 → 输入账号名 → 字号 12-16px → 透明度 15-20%
  → 颜色取画面中最浅的色调 → 放在留白区
```

### 7.3 文案叠加本身就是天然防护

```
治愈系壁纸的核心卖点是"图+文案"的组合

小红书发布版：             付费版：
  带文案版（展示用）         带文案版（高清无水印）
  ────                    纯净版（无文案无水印）
  ↓                        ↓
  用户可以保存              用户得到两个版本
  但文案遮挡了部分画面       按心情切换使用
  且有水印+低清

关键思路：付费买的不只是高清，还有"选择权"——想要纯净版只有付费才有。
```

### 7.4 发布策略：不全量展示

```
壁纸包有10张图：

小红书展示：
  - 只发 3-4 张（精选但不是全部）
  - 发带水印低清版
  - 结尾写"完整10张高清版 → 主页简介"

这样做的好处：
  ✅ 保留了"未公开"内容，付费才有
  ✅ 用户看到3张好看但不够 → 产生购买欲
  ✅ 降低单张图被盗用的价值（不全套）
```

### 7.5 拼图展示代替单张大图

```
闲鱼主图 / 小红书封面：

  ❌ 不要：直接发单张完整大图（最好保存）
  ✅ 应该：拼图展示，每张缩小到原来的 1/4

  ┌─────┐ ┌─────┐
  │ 图1  │ │ 图2  │    每张约 540×960
  └─────┘ └─────┘    保存下来画质很差
  ┌─────┐ ┌─────┐
  │ 图3  │ │ 图4  │
  └─────┘ └─────┘
```

### 7.6 综合防护等级评估

| 防护手段 | 防护效果 | 对内容质量影响 | 推荐度 |
|----------|---------|---------------|--------|
| 缩小分辨率（发布540×960） | ★★★★★ | 低 | ★★★★★ |
| 文案叠加（付费才有纯净版） | ★★★★ | 无（反而增加价值） | ★★★★★ |
| 轻度水印（半透明账号名） | ★★★ | 低 | ★★★★ |
| 只展示部分图（不全套） | ★★★★ | 无 | ★★★★ |
| 拼图展示（非单张大图） | ★★★ | 无 | ★★★ |
